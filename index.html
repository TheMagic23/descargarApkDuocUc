Bienvenido a la aplicación de Duoc UC

En esta página web se podrá realizar la descarga de la aplicación de asistencia Duoc UC

Funcionalidades de la aplicación

Dentro de la aplicación cuenta con distintos apartados

Esta aplicación se creó para poder solucionar los errores que se encuentran a la hora de pasar la asistencia por parte de los docentes

Login

Cuenta con un apartado para loguearse y cuenta con botones para poder registrarse y recuperar contraseña

Registrar

Contiene un apartado para poder registrarse como alumno

Modificar contraseña

Cuenta con un apartado para modificar la contraseña

Inicio Alumno

Cuenta con un apartado para cuando se inicia sesión como alumno

Información

Cuenta con un apartado para poder informarse de cómo se usa esta aplicación

Escaner

Cuenta con un apartado para poder escanear los códigos para la asistencia

Se debe escanear este código para poder descargar esta aplicación

Descargar

Se debe oprimir el botón “Descargar” para por obtener la aplicación
